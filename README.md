# Vim Github Theme

Github color theme for vim.

## Support themes:

- [x] Light default
- [x] Light high contrast
- [x] Light Protanopia & Deuteranopia
- [x] Light Tritanopia
- [x] Dark default
- [x] Dark high contrast
- [x] Dark Protanopia & Deuteranopia
- [x] Dark Tritanopia
- [x] Dark dimmed

## Install

### Download

Vim-Plug

```vim
Plug 'akiicat/vim-github-theme'
```

### Coloring

Full color

```vim
set termguicolors
```

256 Colors

```vim
set t_Co=256
set t_ut=
set notermguicolors
```

16 Colors

```vim
set t_Co=16
set t_ut=
set notermguicolors
```

### Enable in .vimrc

Select one of them

```vim
colorscheme github_dark
colorscheme github_dark_high_contrast
colorscheme github_dark_protanopia_deuteranopia
colorscheme github_dark_tritanopia
colorscheme github_dark_dimmed
colorscheme github_light
colorscheme github_light_high_contrast
colorscheme github_light_protanopia_deuteranopia
colorscheme github_light_tritanopia
```
